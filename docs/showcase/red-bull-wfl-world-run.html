Red Bull WFL World Run

Red Bull brings the energy, we bring the gear

Red Bull runs tight events. The brand standards are high, the timelines are strict, and everything needs to look sharp. For the WFL World Run in Christchurch we provided lighting, audio and rigging for the start/finish zone. The rig had to work for both the pre-race hype and the finish line atmosphere. Punchy PA, stage lighting, and truss structures all went up fast in a public space with limited bump-in time. It's the kind of job where planning makes the difference.
