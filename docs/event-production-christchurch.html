Event Production Christchurch

Full production. Start to finish.

Event production means more than just dropping off some speakers. It's the whole build: site planning, staging, sound, lighting, power, rigging, crew, and making sure everything runs on time. We do this for festivals, concerts, brand activations, and large-scale private events.

We've done production for Red Bull, Rhythm and Vines, Summit Touring shows, Lads Without Labels, and Offline Collective events. The scale changes but the approach is the same. Plan it properly, show up early, build it right, and run it clean.

FAQ

What size events can you handle?

We've done everything from 50-person club nights to multi-day festivals with thousands of attendees. The approach changes based on scale, but we're comfortable at any size.

Do you travel outside Christchurch?

Yes. We're based in Christchurch but we regularly work across Canterbury and around New Zealand. We've done production at Rhythm and Vines in Gisborne and events in other cities. Travel costs are included in the quote.

What's the booking process?

Get in touch with your event details: date, venue, expected crowd, and what you're after. We'll put together a quote and a production plan. Once confirmed, we handle all the logistics from there.
